# Workflow: X To Materials

## 目标

把公开链接或公开内容保存成可复用的素材 Markdown。

## 输入

- 一个公开链接
- 或一段你想保留的公开内容

## 输出

- `materials/source-articles/` 下的新 Markdown 文件

## 推荐执行者

- `boss` 负责决定是否收集
- `material` 负责保存和整理

## 最小步骤

1. 获取公开内容
2. 保存为 Markdown
3. 补充标题、来源、摘要
4. 放进 `materials/source-articles/`

## 完成标准

- 文件可以被别人读懂
- 标题清楚
- 有来源链接
- 有 1 到 3 句摘要
