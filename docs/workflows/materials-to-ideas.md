# Workflow: Materials To Ideas

## 目标

从已有素材里找到值得写的方向。

## 输入

- `materials/index.md`
- 最近新增的素材

## 输出

- `ideas/ideas.md` 里的新条目

## 推荐执行者

- `material` 先给出相关材料
- `thinktank` 给出角度
- `boss` 只保留最值得继续的一条

## 最小步骤

1. 找出相关素材
2. 归纳主题
3. 写一句为什么值得写
4. 记录到 `ideas/ideas.md`

## 完成标准

- 灵感不是空话
- 能看出它基于哪些素材
- 下一步知道该去写大纲
