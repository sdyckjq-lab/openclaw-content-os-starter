# Public Safe Content Contract

这个文件定义什么内容可以进公开仓库，什么绝对不行。

## 可以公开

- 角色职责描述
- workflow 文档
- 模板文件
- 占位符配置
- 通用脚本
- 脱敏后的 skill 说明

## 不可以公开

- 真实密钥
- 真实用户 ID
- 真实群组 ID
- 真实渠道映射
- 私有路径
- 真实姓名、真实邮箱、真实主机名
- 私有素材内容
- 真实客户信息
- 真实 persona 设定
- 内部运维信息

## 发现下面任意一项，就不能发布

- 出现真实手机号、邮箱、用户名
- 出现真实姓名、主机名或可识别的提交身份
- 出现真实 token
- 出现真实 Bot 配置
- 出现你自己的绝对路径
- 出现私有草稿

## 默认规则

拿不准时，宁可不放。
