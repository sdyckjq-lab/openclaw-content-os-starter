# First Prompts

如果你第一次使用这个 starter，不要直接让它做大而全的事情。

先用这些简单提示词：

## 1. 先确认系统结构

```text
请先用一句话解释这个 starter 的 5 个角色分别负责什么。
```

## 2. 先走一遍流程，不要求真正发布

```text
请按默认工作流，演示一次从素材到初稿的步骤，不要做外部发布。
```

## 3. 先从已有内容开始

```text
我有一篇公开文章链接，先帮我把它当成素材，告诉我下一步怎么走。
```

## 4. 只让 boss 作为入口

```text
以后默认由你作为 boss 接收需求，再决定是否分派给其他角色。
```
